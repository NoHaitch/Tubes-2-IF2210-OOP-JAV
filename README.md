## About The Project

Knee Grow is a card game based on Object Oriented Programming using Java. The game is played by 2-4 players.
The game is played by drawing cards from the deck and playing them on the table. The player who has the most points at
the end of the game wins. The game is played in turns, where each player can draw a card from the deck, play a card
from their hand, or draw a card from the discard pile. The game ends when the deck is empty and all players have no
cards left in their hands. The player with the most points wins the game.

This project is made for the final project of IF2210 Object-Oriented Programming course in ITB Informatics Engineering.

### Installation

_How to install and use your project_

1. Clone the repo
   ```sh
   git clone https://github.com/NoHaitch/Tubes-2-IF2210-OOP-JAV
   ```
2. Change Directory
   ```sh
    cd Tubes-2-IF2210-OOP-JAV
    ```
3. Install Maven Dependencies
   ```sh
    mvn install
    ```
4. Run the program
   ```sh
    mvn clean javafx:run
    ```
